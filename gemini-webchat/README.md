# Gemini AI Chatbot 🤖

A conversational AI chatbot built with Python and Google's Gemini API.
Features a custom personality (Aria), conversation memory, and automatic chat history saving.

## Features
- 💬 Multi-turn conversation with memory
- 🧠 Custom AI personality (Aria)
- 💾 Auto-saves chat history to a text file
- ⚡ Powered by Gemini 2.5 Flash model

## Tech Stack
- Python 3.14
- Google Gemini API (google-genai)
- VS Code

## Setup
1. Clone the repo
   git clone https://github.com/borrayaswanth1234-spec/gemini-ai-chatbot.git
2. Install dependencies
   pip install google-genai
3. Add your API key in chatbot.py line 5
4. Run
   python chatbot.py

## Usage
- Type any message and Aria responds instantly
- All conversations saved to chat_history.txt
- Type quit to exit
